# Around The U.S. - Front-end (React + Vite)

Esta es la aplicación cliente de la red social interactiva, donde los usuarios pueden compartir fotos de sus viajes, dar "me gusta" a las publicaciones de otros y gestionar su propio perfil.

## 🛠️ Tecnologías utilizadas

- **React**: Biblioteca principal para la interfaz de usuario.
- **Vite**: Herramienta de construcción rápida para el desarrollo.
- **BEM Methodology**: Para una estructura de CSS organizada y escalable.
- **React Hooks**: Manejo de estado y efectos (useState, useEffect, useContext).

## 🚀 Funcionalidades

- **Gestión de Perfil**: Los usuarios pueden actualizar su nombre, ocupación y foto de avatar.
- **Galería de Fotos**: Visualización de tarjetas con imágenes, títulos y contador de "likes".
- **Interactividad**: Capacidad de añadir nuevas fotos, borrar fotos propias y dar/quitar likes.
- **Validación de Formularios**: Control de entradas de usuario para asegurar datos correctos.
- **Diseño Adaptativo**: Totalmente responsivo para móviles, tablets y escritorio.

## 📦 Instalación y Uso

1. Entra a la carpeta: `cd frontend`
2. Instala dependencias: `npm install`
3. Inicia en modo desarrollo: `npm run dev`
