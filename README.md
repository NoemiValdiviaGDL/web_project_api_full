Estructura del Proyecto
Frontend: Aplicación SPA desarrollada en React. Utiliza hooks, manejo de estado y componentes controlados para una experiencia de usuario fluida y responsiva.

Backend: API RESTful construida con Node.js y Express. Gestiona la lógica de negocio, la persistencia en base de datos, el cifrado de contraseñas y la autorización mediante JWT.

Características Principales
Gestión de Contenido: CRUD completo para tarjetas de fotos (subir, eliminar, dar "like").

Perfil de Usuario: Edición de avatar, nombre y descripción en tiempo real.

Seguridad: Sistema de registro e inicio de sesión con validación de datos.

Diseño Adaptable: Interfaz totalmente responsiva.
